CSS Positions

  1. Static: The original or default position of an element
  2. Relative: Element is positioned relative to its normal/original position. Same a static positioning but here we can move an element by adding offset values.
  3. Absolute: Element is positioned relative w.r.t its parent. If parent is not present, then element is positioned relative w.r.t the main document. Element is removed from the normal document flow. No space is created for the element.
  4. Fixed: Element is postioned relative to the viewport Element doesnt move and stays in place even after scrolling. Element is removed from the normal flow.
  5. Sticky: Element is positioned according to the normal document flow till the element hits the top. After it touches the top, then its position become fixed.
